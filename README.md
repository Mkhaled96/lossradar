# NEXUR FLEET

## اللي جوه المجلد ده
- `src/pages/LoginPage.jsx` — صفحة تسجيل الدخول
- `src/lib/supabaseClient.js` — الاتصال بـ Supabase
- `supabase/schema.sql` — قاعدة البيانات (تم تشغيله بالفعل)
- `supabase/functions/sync-fleet-data` — آلية مزامنة الشيت (تم نشرها بالفعل)
- `.env.example` — مكان مفاتيح Supabase

## خطوة واحدة بس دلوقتي: ارفعه على GitHub

1. روح على github.com وسجل دخول (أو اعمل حساب لو معندكش)
2. دوس **+** فوق يمين → **New repository**
3. سمّيه `nexur-fleet` ودوس **Create repository**
4. في الصفحة اللي هتفتح، دوس على **uploading an existing file**
5. اسحب المجلد ده كامل (أو كل الملفات جواه) وارفعه
6. دوس **Commit changes**

كده المشروع بقى على GitHub. الخطوة اللي بعد كده (ربطه بـ Vercel عشان يبقى موقع شغال) هتيجي في رسالة منفصلة، خطوة بخطوة.
